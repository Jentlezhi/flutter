# Mac端AndroidStudio快捷键

##1. 查看类的结构

```
cmd + 7
```
##2. 查看继承关系
```
control + h
```
##3. 快速创建 StatelessWidget/StatefulWidget
```
stless : 创建一个StatelessWidget
stful:创建一个StatelessWidget
```
